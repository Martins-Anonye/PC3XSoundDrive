Privacy Policy

Effective date: July 6, 2026

This Privacy Policy explains how the BrickRace Music Store (the "Service"), operated by the BrickRace project, collects, uses, and shares information when you use the web admin tools and when the BrickRace game accesses store data. This page covers uploads you make via the admin interface and how the mobile app accesses that data.

Information We Collect

Files and metadata

When an administrator or contributor uploads music files, lyrics, or other media through the Music Store web interface, the Service stores the file and basic metadata (file name, size, upload timestamp, category). Uploaded files are stored in cloud storage (Dropbox or Firebase Storage depending on configuration) and links or metadata are saved to the Firebase Realtime Database.

Authentication information

If you sign in to the admin web interface, we use Firebase Authentication to validate your identity. We do not store raw credentials on this site; authentication is handled by Firebase. The Service may store your user ID and display name as part of upload metadata for audit purposes.

Usage data

We collect basic usage and log information necessary to run the web admin interface and to debug problems (timestamps, IP addresses, browser user-agent strings, and actions such as uploads and deletes). We may also collect analytics-level events if configured.

How We Use Your Data

  • Store and serve uploaded audio and lyric files to the BrickRace mobile app via cloud storage and the Realtime Database.
  • Maintain an audit trail of uploads and removals so admins can manage content.
  • Debug and improve the Service, and monitor for abuse.

Third-Party Services

The Service uses the following third-party providers:

  • Firebase — Authentication, Realtime Database and (optionally) Storage. Please review Firebase's privacy documentation for details about how they process information.
  • Dropbox — Optional storage and file hosting for uploaded media (if configured). Uploaded files may be stored in the connected Dropbox account.

Sharing and Disclosure

Uploaded files and their public share links (if generated) are accessible by anyone with the link. The BrickRace mobile app reads file metadata and download links from the Firebase Realtime Database to present store items to players. We do not sell your personal data.

Data Retention

Files and metadata remain stored until an admin deletes them. Backup and retention policies depend on the configured cloud storage provider (Dropbox/Firebase). You can request deletion using the contact details below.

Security

We take reasonable steps to protect uploaded files and metadata. Access to administrative operations should be restricted to authorized users. However, no internet service is perfectly secure — please avoid uploading sensitive personal data.

Children

The Service is not intended for children under 13. We do not knowingly collect personal information from children under 13. If you believe such information has been uploaded, contact us and we will remove it.

Your Choices

  • Access or deletion: Contact the administrator (see below) to request access or removal of uploaded files or metadata.
  • Opt-out: If you do not want uploaded files to be publicly available, do not provide public share links when uploading. Use private storage options in your cloud provider.

Changes to this Policy

We may update this Privacy Policy from time to time. We will post the revised policy on this page with a new effective date.

Contact

If you have questions or want to request removal of uploaded content, contact: support@yourdomain.com
